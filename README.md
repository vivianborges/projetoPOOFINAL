# Projeto Final – Programação Orientada a Objetos  
## Sistema de Delivery – Java + Swing

Este repositório contém o projeto final da disciplina Programação Orientada a Objetos.  
O sistema simula um aplicativo simples de Delivery, utilizando Java, conceitos de POO e interface gráfica com Swing.

## Estrutura do Projeto

- **controller/** → regras de negócio (PedidoController)
- **main/** → classe principal (Main.java)
- **model/** → classes do domínio (Pessoa, Cliente, Pedido, Calculavel)
- **view/** → interface gráfica com Swing (TelaPrincipal)

## Conceitos de POO Utilizados
- Encapsulamento  
- Herança (Pessoa → Cliente)  
- Polimorfismo  
- Interface (Calculavel)  
- Sobrescrita de métodos  
- Construtores  
- Classes de domínio organizadas  

## Tecnologias
- Java 8+  
- Swing (interface)  
- POO  
- Arquitetura simples MVC  
- GitHub para versionamento  

## Como Executar
1. Baixe o projeto ou clone o repositório  
2. Abra na sua IDE (IntelliJ, Eclipse ou NetBeans)  
3. Execute a classe `Main` em:  /main/Main.java
